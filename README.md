# Web Dev Bootcamp Project

A beginner web development project built with Next.js and Tailwind CSS.

## Tech Stack

- Next.js 15
- TailwindCSS 4
- TypeScript
- Supabase

## Getting Started

```bash
cd my-app
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
